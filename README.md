# Descrição da solução:
Criar uma solução mobile que democratize o acesso a dados científicos de satélites, transformando índices complexos (NDVI, temperatura superficial) em alertas claros e acionáveis para gestores públicos, agricultores e comunidades vulneráveis.

# Como rodar o projeto:
- ```git clone https://github.com/ramalhoalexia/spaceAlert-GS1.git```
- ```npm i```
- ```npm start```
